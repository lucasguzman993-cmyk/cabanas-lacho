# Cabañas Lacho — Sistema de Reservas

## Variables de entorno en Render:
- SECRET_KEY: clave secreta Flask
- ADMIN_USER: usuario admin (default: admin)
- ADMIN_PASS: contraseña admin (default: lacho2024!)
- TIPO_CAMBIO: tipo de cambio USD/ARS (default: 1200)

## URLs:
- / → Landing page
- /cabana/lacho4 → Detalle cabaña 4 personas
- /cabana/lacho6 → Detalle cabaña 6 personas
- /reservar/lacho4 → Formulario reserva
- /admin → Panel administrador
- /admin/login → Login admin
